Helfe bei GORM mit

Du kannst uns dabei helfen, GORM besser zu machen! Es gibt viele Dinge, die du machen kannst:

Hilf anderen dabei, GORM kennenzulernen

  • Teile GORM in Sozialen Netzwerken oder erzähle einfach Menschen, die es interessieren könnte, davon
  • Helfe Anfängern dabei, Fragen auf Stack Overflow, Github Issues und in Chat Räumen zu lösen
  • Schreibe Tutorials, nehme Videos auf, etc. (Vergesse nicht, sie zu der Liste hinzuzufügen!)
  • Helfe dabei, GORM auf GitHub zu bewerben, indem du die GORM Repository mit einem Stern versiehst und sie beobachtest

Dokumentation und Texte

  • Verbessere GORMs Dokumentation, indem du neuen Inhalte hinzufügst und existierenden verbesserst

Übersetze diese Seite

Du kannst die aktuelle Website auf https://translate.gorm.io übersetzen

GORM in deine Sprache zu übersetzen, musst du einen Anfrage, sie auf https://translate.gorm.io hinzuzufügen in den Github Issues posten.

Programm

  • Erstelle einen Pull Request auf Github, um Fehler zu beheben oder um neue, vereinbarte Funktionen aus den Github Issues hinzuzufügen
  • Erstelle Open Source Plugins für GORM. (Vergesse nicht, sie zu der Liste hinzuzufügen!)
